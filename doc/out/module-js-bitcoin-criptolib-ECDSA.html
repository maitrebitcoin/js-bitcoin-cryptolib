Class: ECDSA

js-bitcoin-criptolib~ECDSA()

Main class ecdsa with secp256k1 parameters ex usage : var priv = ecdsa.newPrivateKey(); var pub = ecdsa.publicKeyFormPrivateKey(priv); var signature = ecdsa.signMessage( "my message", priv ) var res = ecdsa.verifySignature( "my message", signature, pub ) if (!res.ok) alert(res.message)
